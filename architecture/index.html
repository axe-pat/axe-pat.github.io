Guardrails

The model is useful because the system knows when to stop it.

G01Exact-run provenance

Reports point to the artifacts produced by that run. A convenient newer file cannot silently replace missing evidence.

G02Human authority

High-cost or irreversible decisions stop at review. The product prepares and recommends; it does not erase operator judgment.

G03Fail-closed execution

Missing company evidence, uncertain delivery, timeouts, or incomplete state block retries and preserve an audit path.

G04Bounded automation

Every live workflow has daily budgets, per-company caps, stop rules, and isolated workers that can be terminated safely.

G05Private-by-design demo

This hosted surface uses aggregate evidence and fictional examples—not real people, applications, messages, or credentials.

G06Advisory learning

Outcomes can propose an experiment, but they never rewrite communication or targeting policy without review.
